# BMI Calculator

## English

This project is a challenge from the [DIO (Digital Innovation One)](https://www.dio.me) platform, developed as part of the Dart learning track.

The goal is to build a command-line BMI (Body Mass Index) calculator that:

- Reads user data (name, weight, height) from the terminal
- Supports two languages: **English** (imperial units) and **Portuguese** (metric units)
- Calculates the BMI based on the selected unit system
- Displays the result with the corresponding classification
- Handles invalid input with exception treatment

### How to run

```bash
dart run
```

### How to test

```bash
dart test
```

## Português

Este projeto é um desafio da plataforma [DIO (Digital Innovation One)](https://www.dio.me), desenvolvido como parte da trilha de aprendizado de Dart.

O objetivo é construir uma calculadora de IMC (Índice de Massa Corporal) via linha de comando que:

- Lê os dados do usuário (nome, peso, altura) pelo terminal
- Suporta dois idiomas: **Inglês** (unidades imperiais) e **Português** (unidades métricas)
- Calcula o IMC com base no sistema de unidades selecionado
- Exibe o resultado com a classificação correspondente
- Trata entradas inválidas com tratamento de exceções

### Como executar

```bash
dart run
```

### Como testar

```bash
dart test
```
